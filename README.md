## Project Description

This is a Single Page Application (SPA) developed for event management. It allows users to register, log in, and interact with the application according to their assigned role: administrator or visitor.

The system includes:

- User authentication and session persistence
- Role-based access (admin and visitor)
- CRUD operations for event management (admin only)
- Event registration for visitors
- Route protection and navigation
- JSON Server for simulating a backend
- Basic and responsive user interface
